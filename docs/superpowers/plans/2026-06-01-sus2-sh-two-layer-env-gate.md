# SUS2-SH Two-Layer Env Gate Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a non-recursive SUS2-SH two-layer neighbor scalar gate where `lambda_b` becomes `lambda_b * (1 + f_b)` and `f_b` is a low-body-order SH scalar fit.

**Architecture:** Keep the existing SUS2 training flow, linear solve, and SH product graph. Add gate metadata and weights to the model file, make gated SH evaluation dispatch to a configuration-level path, and keep legacy per-neighborhood paths unchanged for plain SH models.

**Tech Stack:** C++11, existing Makefile build, SUS2-MLIP `MLMTPR`, existing SH graph generated by `dev_src/sh_model_init.cpp`, remote LSF/MPI test flow on `/work/phy-weigw/20260321_Test`.

feature. After the first 500-configuration benchmark stopped at formal BFGS

gate-chain `d Phi_q^(0) / dx` term and the mixed final-layer tangent term.
Server checks now pass for EFS finite differences and the nonzero-gate
`w_q` loss-gradient block. The fixed 500-configuration, 100-step rerun completed

```bash
python3 - "$plain_pred" "$gated_pred" <<'PY'
import sys
def read(path):
    values = []
    force_columns = None
    with open(path) as handle:
        for raw in handle:
            line = raw.strip()
            if line.startswith("Energy"):
                values.append(float(line.split()[1]))
            elif line.startswith("AtomData:"):
                header = line.split()[1:]
                force_columns = [header.index(name) for name in ("fx", "fy", "fz")]
            elif force_columns is not None and line[:1].isdigit():
                parts = line.split()
                values.extend(float(parts[idx]) for idx in force_columns)
    return values
plain, gated = read(sys.argv[1]), read(sys.argv[2])
assert len(plain) == len(gated)
err = max(abs(a-b) for a, b in zip(plain, gated))
assert err < 1e-10, err
PY
```

- [x] **Step 3: Add reverse accumulation**

For each final scalar, reverse through the final SH product graph to basic final moments. Each edge contribution accumulates direct coordinate derivatives and a `gate_adjoint[neighbor]` term:

```text
dL/df_b += dL/dM_i^(1)(a) * lambda_a lambda_b R_i(r_ab) Y_i(rhat_ab)
```

Then reverse each atom-local gate scalar:

```text
dL/dw_q += dL/df_b * Phi_q^(0)(b)
dL/dPhi_q^(0)(b) += dL/df_b * w_q
```

Backpropagate the gate scalar adjoints through the unmodulated graph and first-layer basic moments.
